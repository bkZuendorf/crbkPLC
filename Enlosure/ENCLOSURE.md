# Enclosure
A CAD solution in Fusion 360 for a simple and suitable housing for the top-hat rail.

A ready to 3D-print file is provided [in 3mf format](crbkPLCEnclosure.3mf).
In case you're interestet in the Fusion360 construction file feel free to contact me directly.
